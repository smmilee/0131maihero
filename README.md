# 🍋 귀여운 레몬톤 스네이크 게임  
**Cute Lemon Pixel Snake Game**

작고 귀여운 **레몬 파스텔톤 스네이크 게임**입니다.  
뱀은 **동글동글한 픽셀 스타일**, 먹이는 **사과 🍎**, 전체 UI는 **밝고 산뜻한 레몬톤**으로 디자인되어 있습니다.

---

## ✨ 특징 (Features)

- 🍋 레몬 파스텔톤 UI  
- 🐍 동글동글한 픽셀 스타일의 귀여운 뱀  
- 🍎 픽셀 사과 먹이  
- 🎮 키보드 화살표 키로 조작  
- 📄 HTML 단일 파일 (CSS + JS 포함)  
- 🚀 브라우저에서 바로 실행 가능  

---

## 🎮 플레이 방법 (How to Play)

- **↑ ↓ ← →** 화살표 키로 뱀을 조작합니다.  
- 사과 🍎를 먹으면 점수가 올라가고 뱀이 길어집니다.  
- 벽이나 자기 몸에 닿으면 게임 오버!  

---
